## [示例] 2026-06-09 Windows GBK 编码导致脚本崩溃
- **场景**：在 Windows 跑 audit 脚本读取含中文的 SKILL.md
- **问题**：UnicodeDecodeError 直接崩溃
- **根因**：默认按 GBK/locale 解码 utf-8 文件
- **修正**：读取统一走 utf-8 → utf-8-sig → gbk → errors=replace 回退链
- **规则**：所有读文件函数必须用多编码回退，禁止裸 open()
